# Phase 0: Discovery

**Objective:** Read all forecasting system files, assess current state, and write a persistent discovery summary to disk. This summary is the context source for all subsequent phases.

**Do not write any code during this phase other than the discovery summary file.**

---

## D1. Read Architecture Documents

Read the following files. If any exceeds 500 lines, read the first 200 and last 200 lines plus section headers.

